## 프로젝트 정보 

### 1.프로젝트 주제

서울 음식점 추천 사이트

### 2.주제 선정배경 또는 이유

점심시간 때  음식점 고르는 시간을 해소하기 위해 음식점 추천 서비스를 제공하고자 합니다.

### 3. 프로젝트 개요

가게 선택후 후기와 별점을 남길 수 있고, 음식 추천 서비스을 이용해 음식 메뉴를 고를 수도 있습니다.
